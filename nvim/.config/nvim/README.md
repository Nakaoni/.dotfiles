# nvim.config
Configuration files

Based on [LunarVim configuration](https://github.com/LunarVim/Neovim-from-scratch)

## Requirements:
- Neovim >=0.8
- tree-sitter >= 0.20
- ripgrep >= 13.0
- node >=14.0
- composer >=2.0
- gcc/g++ >= 11.0
- git >= 2.34

## NerdFonts Icons
To install Fonts, run:
```sh
./install_fonts.lua
```
